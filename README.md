# OuterLi Blog

OuterLi 的个人博客，已全量迁移到 [joyehuang/blog](https://github.com/joyehuang/blog) 开源模板底座。

## Stack

- Astro 5
- Astro Theme Pure
- UnoCSS
- MDX
- Cloudflare Pages static output

## Commands

```bash
npm install
npm run dev
npm run check
npm run build
```

Cloudflare Pages:

- Build command: `npm run build`
- Output directory: `dist`
